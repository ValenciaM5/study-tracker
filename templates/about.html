About Study Tracker

Study Tracker is a small Go web application I built as my own end-to-end DevOps practice project. It runs locally first, and from there I'm taking it through every step of the modern DevOps workflow — Docker, Kubernetes, Helm, GitHub Actions, and Argo CD.

Built with the Go standard library — no frameworks, just net/http and html/template.
